
# 🕒 Digital Clock

A sleek, modern digital clock built with HTML, CSS, and JavaScript.  
Includes real‑time display, 12/24‑hour format toggle, light/dark theme, and a fully responsive design.

🔗 **Live demo:** [digital-clock.vercel.app](https://digital-clock.vercel.app)

---

## ✨ Features

- ⏱️ Real‑time updating hours, minutes, and seconds  
- 🔁 Switch between **12h / 24h** format  
- 🌓 Toggle between **light and dark themes**  
- 📅 Current date displayed in English  
- 📱 Fully **responsive** – works on desktop, tablet, and mobile  
- 🎨 Modern glass‑morphism design with smooth transitions

---

## 🛠️ Built with

- **HTML5** – semantic structure  
- **CSS3** – Flexbox, media queries, backdrop‑filter, custom properties  
- **JavaScript (ES6)** – DOM manipulation, event listeners, `setInterval`
